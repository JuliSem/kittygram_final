### Как запустить проект:

Клонировать репозиторий и перейти в него в командной строке:

```
git clone "адрес клонируемого репозитория"
```

```
cd kittygram_final
```

Перейти в папку с фронтендом:

```
cd frontend
```

Установить зависимости:

```
npm i
```

Запустить проект:

```
npm run start
```
